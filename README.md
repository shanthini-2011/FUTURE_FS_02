A simple, elegant, and fully functional Mini CRM built to manage client leads. This project features a modern 3D Glassmorphism UI and a robust backend supporting complete CRUD operations.

## ✨ Key Features
- **Secure Admin Access:** Protected dashboard entry using local storage session validation.
- **Lead Listing:** View all leads with detailed information (Name, Email, Source, Status).
- **Status Updates & Notes:** Seamlessly update lead progress (New / Contacted / Converted) and add follow-up notes.
- **Full CRUD Operations:** Create new leads, Read existing ones, Update details, and Delete unwanted leads from the database.
- **Modern UI/UX:** Premium 3D glassmorphism design, interactive hover animations, and real-time toast notifications.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** SQLite
